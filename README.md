# Attendance AI - Face Recognition Attendance System

## 📌 Overview

Attendance AI is a Machine Learning and Computer Vision project that automates attendance management using facial recognition. The system captures faces through a webcam, identifies registered users, and records attendance automatically with timestamps.

This project eliminates manual attendance marking, improves accuracy, and saves time in educational institutions and workplaces.

---

## ✨ Features

* Face Detection using OpenCV
* Face Recognition for user identification
* Automated Attendance Recording
* User Registration Module
* Model Training Functionality
* Real-Time Webcam Integration
* Attendance Storage in CSV Format
* Simple and User-Friendly Interface

## 🚀 How It Works

1. Register a new user by entering details and capturing facial images.
2. Train the recognition model using the collected images.
3. Start the attendance system.
4. The webcam detects and recognizes faces in real time.
5. Attendance is automatically marked and saved.
